# LoRaWAN Door Sensor

## Overview
The Dragino LDS03A is an outdoor rated LoRaWAN Door Sensor. It detects door open/close status and uplink to IoT
server via LoRaWAN network. user can see the door status, open time, open counts in the IoT
Server.
